API Request Flow

Overview: Incoming prediction requests are validated, routed to appropriate models, scored with walk-forward thresholds, and returned with confidence scores - all in <500ms.
sequenceDiagram participant Client participant API as BETBOT API participant Router as Model Router participant Cache as Threshold Cache participant DB as MongoDB participant Model as ML Model participant Tracker as BETTRACKER Client->>API: POST /predict {entity_id, features} API->>API: Validate request (Pydantic) API->>Router: Route by market category Router->>Cache: Check threshold cache alt Cache Hit (75% of requests) Cache->>Router: Return cached threshold else Cache Miss Router->>DB: Fetch past 180 days DB->>Router: Historical predictions Router->>Router: Optimize threshold (101 candidates) Router->>Cache: Store threshold end Router->>Model: predict_proba(features) Model->>Router: confidence = 0.58 Router->>Router: Apply threshold: 0.58 >= 0.42 Router->>API: {prediction: 1, confidence: 0.58} API->>Client: 200 OK + prediction bundle API->>Tracker: Log prediction async Note over Client,Tracker: Total latency: <500ms

Walk-Forward Threshold Optimization

Temporal Rigor: Thresholds are recalculated using only historical data available before prediction time. 101 candidates evaluated with 75% cache hit rate for efficiency.

Hot-Swap Model Update

Zero-Downtime Deployment: Update models in production without service interruption using atomic pointer swaps.
sequenceDiagram participant Admin participant API as BETBOT API participant Memory as Memory participant OldModel as Old Model participant NewModel as New Model (Loading) participant Requests as Incoming Requests Admin->>API: POST /models/hot-swap {model_path} API->>Memory: Load new model to staging Memory->>NewModel: Initialize NewModel->>NewModel: Validate model format NewModel->>Memory: Ready Note over Requests,OldModel: Requests still use old model Memory->>Memory: Atomic pointer swap Memory->>API: Swap complete API->>Admin: 200 OK - Model updated Note over Requests,NewModel: New requests use new model Memory->>OldModel: Garbage collect OldModel-->>Memory: Freed Note over API: Total downtime: 0ms
